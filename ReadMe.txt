Hello team steps use this application
1)There are two projects sudoku client app developed in angular 8 and sudoku api developed in asp.net core api
2) For client app you can provide manual input via json file called input.json.
3) Open Sudoku api solution and press F5, after that open sudoku client in VS code or any editor serve the angular application.
 to enjoy solving sudoku.
